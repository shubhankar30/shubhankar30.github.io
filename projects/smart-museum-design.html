Smart Museum Design based on Indoor Localization

Publication:

"Co-Author,"Smart Museum Design based on Indoor Localization", IJCSN Journal Volume 5, Issue 2

Publication link

Project Report



.In this project,we have proposed an indoor localization technique for a scenario in a museum environment. Moreover after estimating the position, context based information will be provided to the user.

The proposed system is based upon the fingerprinting technique. Initially in the training or the mapping phase, the RSSI values of candidate locations are recorded in the database. In the second phase the RSSI values are taken as input and using a suitable algorithm the approximate position of the user is determined.
